## Обработка пропусков

Использовался `SimpleImputer`:

- `strategy="constant"` — замена пропусков на "Unknown"
- `strategy="median"` — для числовых признаков (устойчиво к выбросам)

Важно:
- `fit()` выполняется только на обучающей выборке
- `transform()` применяется к тестовой

# Часть 3 — Масштабирование признаков

Рассмотрены два способа масштабирования:

## MinMaxScaler
Приводит значения к диапазону [0, 1].

Используется, когда важен фиксированный диапазон значений.

---

## StandardScaler
Стандартизирует данные:
(x − mean) / std

Используется, когда модели чувствительны к масштабу признаков.
